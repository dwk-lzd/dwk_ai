git status  查看状态  （本地）
git init  初始化仓库  （本地）
git add 文件名  把文件添加到暂存区  （本地）   git add .  把所有修改文件添加到暂存区
git diff 文件名  查看文件的修改  （本地）
git log --oneline  查看提交历史  （本地）
git commit -m "修改描述"  提交到仓库  （本地）   
git push origin main  把本地仓库推送到远程仓库  （本地）
